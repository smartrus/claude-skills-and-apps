# Whimsy Injector

This skill identifies opportunities to add personality, delight, and playful moments to user experiences. It audits interfaces and user journeys to find friction points where micro-interactions, Easter eggs, and witty copy can reduce anxiety and increase emotional engagement.

## When to Use

Use this skill when:
- Auditing sterile or anxiety-inducing workflows
- Designing micro-interactions for specific moments
- Creating Easter eggs and hidden delights
- Adding personality and humor to error states
- Improving emotional responses in boring flows
- Considering surprises that don't harm task completion
- Balancing playfulness with accessibility requirements

## When NOT to Use

Do NOT use this skill for:
- Comprehensive UI/UX design overhauls
- WCAG accessibility auditing (use accessibility-wcag-auditor)
- Brand voice and tone establishment
- Gamification strategy and mechanics
- Animation/motion design outside of delight context
