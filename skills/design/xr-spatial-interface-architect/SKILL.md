# XR Spatial Interface Architect

This skill specializes in designing user interfaces and interaction patterns for extended reality (XR) environments including VR, AR, and mixed reality systems. It produces comprehensive interaction design specifications that account for the unique constraints and affordances of spatial computing.

## When to Use

Use this skill when designing:
- 3D spatial UI layouts for VR applications
- Hand gesture vocabularies and interaction models
- Gaze-tracking based selection and navigation
- Spatial audio design for UX feedback
- Comfort zone mapping to prevent simulator sickness
- Locomotion patterns for 3D navigation
- Multi-modal interaction (hand, gaze, voice, controllers)
- Platform-specific design constraints (Meta Quest, Vision Pro, HoloLens, etc.)

## When NOT to Use

Do NOT use this skill for:
- Traditional 2D web or mobile app interface design
- General UX research without XR-specific context
- Immersive storytelling or narrative design
- Game design and gameplay mechanics
- 3D modeling, rendering, or graphics programming
