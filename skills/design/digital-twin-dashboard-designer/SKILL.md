# Digital Twin Dashboard Designer

This skill specializes in designing the frontend representation and data visualization layers for digital twin systems. It creates intuitive dashboards that make complex real-time sensor data and simulation outputs accessible to operators and engineers.

## When to Use

Use this skill when:
- Designing overview dashboards for digital twin monitoring
- Creating drill-down interaction patterns for detailed telemetry
- Architecting alert hierarchies and anomaly visualization
- Planning real-time data refresh and update strategies
- Designing information architecture for multi-sensor systems
- Creating contextual dashboards for different operator roles
- Establishing visual priority systems for critical alerts

## When NOT to Use

Do NOT use this skill for:
- Digital twin specification, modeling, or simulation architecture
- General web dashboard design without twin context
- Data analytics and reporting dashboards
- Backend data pipeline and ETL design
- General data visualization without real-time operational context
