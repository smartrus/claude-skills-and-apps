# Contributing

See the full [CONTRIBUTING.md](https://github.com/smartrus/claude-skills-and-apps/blob/dev/CONTRIBUTING.md) on GitHub for complete guidelines.

## Quick Reference

1. Fork and clone the repo
2. Branch from `dev` (never `main`)
3. Use `./scripts/new-skill.sh` or `./scripts/new-app.sh` to scaffold
4. Run `./scripts/lint.sh` before committing
5. Open a PR targeting `dev`

## Commit Format

```
feat(engineering): add pr-review skill
fix(data): correct SQL dialect detection
docs: update contributing guide
```
